# 🏸 NaDream & Bank - 3D Badminton Birthday RPG 🎂✨

An interactive 3D Web Game built with **HTML5**, **CSS3**, **JavaScript**, and **Three.js** for NaDream's Birthday!

## 🎮 Game Highlights
- **Spacious Indoor Badminton Complex**: 5 Rooms featuring 4 authentic BWF green badminton courts per room with realistic white boundary lines.
- **Camera-Relative Controls**: Smooth movement turning where the camera points, natural walking speed, and jump physics.
- **Cinematic Secret Door Cutscenes**: Secret doors slide open with radiant light beam flares when completing room checkpoints 1 to 5.
- **NPC Bank & Finale**: Meet NPC **Bank** in the Grand Winner's Sanctuary holding a badminton racket for the Birthday greeting!
